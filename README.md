# Storefront Backend Project

## Commands:

### Install Packages:
`npm i`

### Run the server:
`npm run start`

### run the server on watch:
`npm run watch`

### test the server:
`npm run test`

### Database migration:
`db-migrate up`

### Start postgres Docker:
`docker-compose up`
